搭建Hexo博客

Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Heroku上，是搭建博客的首选框架。

一、操作概述

  • 购买个人域名（可选步骤）

  • 创建GitHub个人仓库

  • 安装环境(安装Git、Node.js)

  • 安装及配置Hexo

  • Hexo与Github仓库连接

  • 绑定个人域名

二、详细步骤

1.购买个人域名

​ 本教程是通过阿里云购买的个人域名，读者可以根据个人情况自行选择。另外，此项是本教程搭建博客唯一需要支出的地方，并且如果不需要更改自定义域名，使用github pages域名即可。

2.注册github账号，创建仓库

​ （建立github仓库，格式必须为：username.github.io）

  • 生成ssh密匙

    ssh-keygen -t rsa -C "example@mail.com"

    blog目录下，git命令行输入，然后直接三个回车即可，默认不需要设置密码。

    然后在硬盘中找到id_rsa.pub密钥文件，将内容全部复制到GitHub_Settings_keys页面，新建New SSH Key，粘贴所有内容。

创建成功后在git命令行输入ssh git@github.com

若成功创建，则显示

3.安装环境

  • 安装git

    安装成功后，鼠标右键选单会出现Git Bash here

  • 安装Node.js

    （node -v\ npm -v查看是否安装成功）

4.安装以及配置hexo

创建博客根目录文件夹，通过git bash执行以下内容

npm install -g cnpm --registry=https://registry.npm.taobao.org
npm install hexo --save --registry=https://registry.npm.taobao.org

​ 通过npm安装hexo，安装成功后使用hexo -v查看版本

hexo init

​ 初始化Hexo（需要在空目录下执行该命令）

npm install --registry=https://registry.npm.taobao.org

​ 补全依赖环境

hexo g

​ 生成静态页面

hexo s

​ 开启本地预览

​ 然后打开浏览器，输入 localhost:4000 就可以预览我们的本地博客

​

5.Hexo与Github仓库连接

  • blog根目录中的_config.yml文件称为站点配置文件，修改为如下内容

    deploy: 
    type: git
    repository: https://github.com/username/username.github.io.git
    branch: master

  • 设置用户信息

    git config --global user.name "shiroineil"
git config --global user.email "keepermx69@gamil.com"

    可通过git config --list来检验用户名和邮箱是否设置成功

  • 安装npm的发布拓展

    npm install hexo-deployer-git --save --registry=https://registry.npm.taobao.org
    hexo g
    hexo s
    hexo d

    发布页面

    命令 含义
    hexo n “title” 新建一篇文章
    hexo g g=generate 生成静态页面至public目录
    hexo s s=server 开启预览访问端口
    hexo d d=deploy 发布至GitHub
    hexo -v v=version 显示hexo版本信息

6.绑定个人域名

  • 阿里云网页内，域名解析—>解析设置—>选择域名—>添加记录

    更改为如图内容

  • 添加记录详细：

    注：1. 记录类型设置为”CNAME“或”A“，主机记录设置为”www”和”@”

    ​ 2. CNAME对应记录的内容为需要绑定的域名，A对应ipv4地址

  • github仓库设置中，添加个人域名
  • 注意：避免每次发布后网页404，要在Blog\public目录下创建CNAME文件(无后缀)，其内容为仓库域名 （域名不需要添加http）

7.Start to Blog

个人博客已经搭建完毕，那么，该如何写文章呢？

发布文章

hexo n "my first Blog"
INFO  Created: D:\Amusement\Blog\source\_posts\my-first-Blog.md

​ 在blog根目录下的source文件夹中的_post文件夹中多了一个my first Blog.md的文件，开始用markdown写博客吧！

​ 写完文章保存后，通过命令hexo s --debug在本地浏览器localhost:4000预览效果，确认无误后，通过hexo g、hexo d 生成、部署网页。随后可以在浏览器中输入域名浏览。

三、花里胡哨(待补充)

  • 更改主题

    可以通过Hexo官网下载相关主题

    1. 单击主题标题进入github页面

  1. 下载压缩文件，然后解压至博客根目录的theme文件夹内

  2. 修改博客根目录下_config.yml文件内容

    ​ 注意：冒号后有空格

  • 设置背景

    请阅读主题相关文档进行更改

  • 增加侧栏菜单条目

    默认的侧栏菜单条目有：首页、归档、标签、关于、搜索等。如果你想要增加其他的菜单条目，修改主题配置文件_config.yml里的Menu Settings中的menu和menu_icons两个地方

  • 寻找图床

    推荐imgchr

  • 添加插件

    • 网易云音乐

      1.打开网页版的网易云音乐，选择喜欢的音乐，点击生成外链播放器

      2.复制外链的HTML代码

      3.比如在侧栏插入这首歌的音乐播放器，修改blog\themes\next\layout_macro的sidebar.swig文件，添加刚刚复制的外链代码

  • 浮动live2d

    原作者EYHN ，现有live2d插件预览

    1).首先安装模块

    npm install --save hexo-helper-live2d --registry=https://registry.npm.taobao.org

    2).文件配置

    请向Hexo的 _config.yml 文件或主题的 _config.yml 文件中添加配置.

    live2d:
  enable: true
  scriptFrom: local
  pluginRootPath: live2dw/
  pluginJsPath: lib/
  pluginModelPath: assets/
  tagMode: false
  debug: false
  model:
    use: live2d-widget-model-koharu
  display:
    position: left
    width: 150
    height: 300
  mobile:
    show: true
  react:
    opacity: 0.7

    3).安装模型

    npm install live2d-widget-model-koharu --registry=https://registry.npm.taobao.org
